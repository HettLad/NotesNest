# 📚 NotesNest – Online Notes Sharing Platform

NotesNest is a full-stack web application that allows students to upload, manage, and download academic notes efficiently. It includes separate user and admin panels with secure authentication and role-based access control.

---

## 🚀 Live Demo

🔗 User Panel: https://notesnest.infinityfree.me/  
🔗 Admin Panel: https://notesnest.infinityfree.me/admin/  

> Demo credentials available upon request

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server:** InfinityFree (Hosting)

---

## ✨ Features

### 👤 User Features
- User registration & login system  
- Upload and download notes  
- Subject-wise categorization  
- Secure authentication and session management  

### 🛠️ Admin Features
- Admin login panel  
- Manage users and uploaded notes  
- Monitor platform activity  
- Control and organize content

## ⚙️ Installation & Setup

1. Clone the repository
  git clone https://github.com/HettLad/NotesNest.git
2. Move project to your server directory (XAMPP/htdocs)
3. Import database
  Open phpMyAdmin
  Create a database
  Import the provided .sql file
4. Update database configuration
  $conn = mysqli_connect("localhost", "root", "", "your_db_name");
5. Run project in browser
  http://localhost/NotesNest
