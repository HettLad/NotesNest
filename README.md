# NotesNest
NotesNest is a PHP &amp; MySQL based notes-sharing platform where anyone can upload, view and share academic notes by subjects and tags. I developed the full backend with authentication, file upload, database handling and partly worked on the frontend UI for smooth user experience.

## Installation
1. Clone the repository
2. Import `Database/notesnest_db.sql` into phpMyAdmin
3. Run project in localhost (XAMPP/WAMP/Laragon)

## Requirements
- PHP 7+
- MySQL
- Apache Server
